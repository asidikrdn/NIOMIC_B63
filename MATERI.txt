PERTEMUAN 1 : Apa itu HTML

HTML
HTML merupakan singkatan dari Hypertext Markup Language. Singkatan ini terdiri dari 3 komponen kata, yakni: Hypertext, Markup dan Language yaitu bahasa markup standar yang digunakan untuk membuat halaman web.
HyperText Markup Language (HTML) adalah bahasa inti dari hampir semua konten Web. Sebuah bahasa yang menggambarkan struktur dan isi semantik dari sebuah dokumen Web. Sama seperti yang dijelaskan dalam wikipedia tentang HTML : https://id.wikipedia.org/wiki/HTML
Saat menjelaskan apa itu HTML, kita tidak bisa melepaskan dengan CSS (Cascading Style Sheets) dan JS (Javascript).  Untuk membuat sebuah website kita tidak bisa memisahkan antara ketiga hal tersebut yaitu HTML, CSS dan JS.

Untuk mempermudah dalam memahami apa itu HTML, mari kita buat sebuah perumpamaan seperti membangun sebuah rumah ?
    HTML    : Diibaratkan seperti membangun sebuah rumah maka HTML itu adalah kerangkanya saja, hanya kerangka sebuah rumah
    CSS     : Akan memiliki fungsi untuk mengatur warna, ketinggian, panjang, atau lebar dari sebuah pintu, jendela, dinding dll. Intinya CSS memiliki fungsi untuk mengatur semua warna, bentuk ataupun ukuran dari panjang lebarnya sebuah rumah.
    JS      : Akan memiliki fungsi untuk mengatur pintu rumah dibuka kedalam / keluar, mengatur saluran air, mengatur instalasi listrik seperti Ketika di Tekan ON maka lampu kamar menyala.

PERTEMUAN 3 : Struktur HTML

Kita akan memahami istilah-istilah dasar pada HTML yaitu :
    1. Element HTML
        Jika kita bilang Element, maka yang dimaksud adalah satu bagian utuh yang terdiri dari tag, atribut, dan seluruh teks yang berada diantara tag pembuka dan penutup.
        namun untuk mempermudah memahami Element, berikut adalah beberapa contoh dari element :
        - Element Meta      : <meta charset="utf-8">
        - Element HTML      : <html></html>
        - Element Head      : <head></head>
        - Element title     : <title></title>
        - Element body      : <body></body>
    
    2. Tag (Pembuka dan Penutup) dan Self Closing Tag 
        Tag adalah bagian dari sebuah element, tag sendiri ada yang disebut : tag pembuka, tag penutup, dan self Closing tag.
        Kita akan coba memahami ketiga hal tersebut. 
        - Tag Pembuka
            Pada element body, yang disebut tag pembuka adalah      : <body>
            Pada element tittle, yang disebut tag pembuka adalah    : <tittle>
        - Tag Penutup
            Pada element body, yang disebut tag penutup adalah      : </body>
            Pada element tittle, yang disebut tag penutup adalah      : </tittle>
        - Self Closing Tag
            Self Closing Tag adalah element dalam html yang tidak memiliki pasangan tag pembuka dan penutup.
            Contoh  : <meta charset="utf-8">

            Ada banyak element yang sifatnya self closing tag seperti :
            <area />   <base />   <br />   <col />   <command />   <embed />   <hr />   <img />   <input />
            <keygen />   <link />   <menuitem />   <meta />   <param />   <source />   <track />   <wbr />

            Penulisan "/" pada self closing tag dalam HTML 5 bersifat optional, dan lebih baik ditinggalkan. 
            Sehingga nanti ketika kita belajar menuliskan element yang sifatnya self closing tag, kita cukup menuliskannya seperti ini :
            <meta>, <img>, <hr>, <br>, dan seterusnya.

            Namun sekarang kita tidak akan membahas fungsi dari element-element tersebut, kita saat ini hanya perlu memahami istilah-istilah yang berada di dalam element HTML.

    3. Attributes
        Attribute adalah informasi tambahan yang ditulis pada tag pembuka. 
        Fungsi dari atribut ini bermacam-macam tergantung nilai dan pada tag mana ia ditempatkan.
        Untuk memahami atribut maka kita akan menuliskan contoh element yang didalamnya memiliki atribut :
        
        <a id="info" href="https://niomic.com">Info Pembelajaran HTML</a>

        Ket :
        - Element a         : <a id="info" href="https://niomic.com">Info Pembelajaran HTML</a>
        - Tag Pembuka       : <a id="info" href="https://niomic.com">
        - Tag Penutup       : </a>
        - Attribute         : id="info"
                              href="https://niomic.com"
            Nama Attribute  : id dan href
            Value/Nilai     : "info" dan "https://niomic.com"

    4. DOCTYPE / DTD (Document Type Declaration)
        DOCTYPE atau Document Type Declaration (DTD) adalah sebuah keterangan yang ditulis untuk memberitahu web browser tentang aturan penulisan dari dokumen yang sedang ditampilkan. 
        Doctype ditulis pada baris pertama halaman HTML.
        Pada HTML versi 4.01, DOCTYPE ditulis seperti ini :
        <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
        "https://www.w3.org/TR/html14/strict.dtd">

        Pada HTML 5 Penulisan doctype jauh lebih sederhana yaitu : <!DOCTYPE html>
    
    5. Body Element
        Di dalam element body inilah tempat kita akan menuliskan code-code HTML yang akan kita tampilkan pada web browser

PERTEMUAN 6 : HEAD Element

Element yang sering digunakan dan diletakkan di dalam element head :
1. Element Meta
    Tag <meta> berfungsi untuk memberi informasi mengenai halaman HTML yang saat ini sedang ditampilkan
    - Meta charset  : berfungsi memberi instruksi kepada web browser tentang bagaimana cara menerjemahkan karakter-karakter yang ada di dalam halaman HTML.
    Pada HTML 4 meta charset ditulis seperti ini : <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    Namun pada HTML 4 penulisannya jauh lebih ringkas : <meta charset="UTF-8">
    Ketika kita menuliskan struktur HTML dan didalamnya kita letakkan meta caharset ataupun tidak, kita tidak bisa melihat perbedaannya karena meta charset ini sendiri berhubungan bagaimana web browser memproses pada karakter-karakter yang berada di dalam HTML.

    - Meta Author, Keywords dan Description
        Ketika kita menuliskan Meta Author, Keywords, dan Description maka ketiga meta tersebut berhubungan dengan SEO (Search Engine Optimization) yaitu digunakan untuk mempermudah mesin pencari seperti google untuk mendapatkan informasi dari website yang kita buat.
        Sehingga ketika kita memasang ketiga meta tersebut di dalam struktur HTML kita, kita tidak bisa melihat perbedaannya karena itu berhubungan dengan web browser memberikan informasi kepada 'robot' / mesin pencari.
        Contoh Penulisan : 
        * <meta name="author" content="Makanan Nusantara">
        * <meta name="keywords" content="Rendang, Bubur Menado, Gulai Nangka Muda, Lontong Tahu, Soto Makasan">
        * <meta name="description" content="Situs Tentang Makanan Nusantara">

    - Meta Refresh
        Meta refresh digunakan untuk melakukan web refresh / reload pada halaman website kita, Contoh : jika kita ingin web refresh / reload website kita setiap 60 detik sekali, maka kita cukup menuliskan code meta refresh.
        <meta http-equiv="refresh" content="60">
        
    -Meta tag Viewport
        Meta Viewport yang ditulis seperti ini : <meta name="viewport" content="width=device-width, initial-scale=1">
        Digunakan pada web dengan desain responsive (Responsive Web Design). Secara garis besar, mea tag viewport ditambahkan agar desain responsive dapat tampil sempurna di dalam layar berukuran kecil (seperti tablet atau smartphone).
        Meta tag viewport ini nanti akan kita praktekkan saat belajar dengan web responsive di materi selanjutnya.
        
    NOTE : Untuk Meta tag X-UA-Compatible dan Meta tag Robots, saat ini bekum diterapkan karena hal ini sudah masuk ke tingkat teknis, jika nanti sudah mampu membuat sebuah website dengan HTML silahkan kembali untuk mempelajari kedua meta tag tersebut.

    - Meta tag X-UA-Compatible
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        Digunakan untuk menangani permasalahan terkait web browser Internet Explorer. IE memiliki mode compatibility view yang aktif ketika halaman web diakses dari komputer lokal atau diakses dari dalam Intranet.
        Fitur ini membuat perlakuan berbeda dalam menangani halaman web (mirip quirks mode).
        Meta tag diatas akan memaksa IE untuk tidak masuk ke mode compatibility view.

    - Meta tag Robots
        <meta name="robots" content="index, follow"> atau <meta name="robots" content="noindex, nofollow">
        Digunakan untuk memberi pesan kepada search engine bagaimana halaman web akan diproses, jika nilai atribut content adalah "index, follow", maka mesin pencari akan mengindex halaman tersebut (dimasukkan kedalam hasil pencarian).
        Namun jika nilai atribut content adalah "noindex, nofollow", maka isi web tidak akan dimasukkan ke hasil pencarian (mungkin halaman web bersifat rahasia dan anda tidak ingin seseorang mengaksesnya dari google).
    
2. Element Title
    Element title berfungsi untuk menampilkan judul pada halaman website yang kita buat. 
    Contoh : 
    <title>Belajar HTML - NIOMIC</title>

    Jika dijalankan maka tab pada browser saat membuka file HTML kita akan tampil tulisan sesuaai dengan isi dari element title.

3. Element Link
    Element link akan sering kita gunakan ketika kita berhubungan dengan file lain, contoh ketika kita ingin menghubungkan HTML kita dengan file css,
    codenya : <link href="style.css" rel="stylesheet">
    Kita akan membahas hal tersebut nanti saat kita memasuki pembelajaran CSS.

    Contoh menggunakan element link untuk membuat Favicon (icon yang ditampilan pada tab browser)
    <link rel="icon" href="http://www.iconarchive.com/download/i97515/sicons/basic-round-social/facebook.ico" type="image/x-icon">
    - Gambar dari internet
        Pada contoh diatas, kita menampilkan Favicon dari internet yaitu melalui link icon tersebut
    - Gambar dari local
        Jika kita meletakkan gambar dalam folder tempat kita menuliskan code HTMLnya
        <link rel="icon" href="namaicon.png" type="image/png">
        Jika gambar kita letakkan di dalam folder lagi, misal folder img, maka seperti berikut
        <link rel="icon" href="img/namaicon.jpg" type="image/jpg">
        
    NOTE :
    pada atribut type="image/png" kita bisa menggantinya sesuai dengan gambar yang kita masukkan, jika gambar yang dimasukkan adalah jpg maka atribut type menjadi type="image/jpg",
    dan jika gambarnya berformat .ico maka menjadi type="image/x-icon".
    
4. Element Base
    Element base bersifat global, dan digunakan untuk memasukkan sebuah link secara global.
    Element base ini akan kita coba praktekkan saat kita belajar materi HTML selanjutnya, karena element base berhubungan dengan element lain, agar ketika dipraktekkan menjadi lebih paham.
    Element base ditulis seperti ini : <base href="/media/images" target="_blank">
    
5. Element Script
    Element Script digunakan untuk menginput Script kedalam halaman HTML, element script sering digunakan untuk menuliskan code Javascript, baik yang dituliskan secara internal ataupun menghubungkan secara eksternal.
    Code ditulis seperti ini : <script type="text/javascript"></script>
    Element script akan kita praktekkan nanti saat kita mulai belajar javascript.

6. Element Style
    Element Style digunakan untuk menginput kode style ke dalam halaman HTML.
    Element Style ditulis seperti ini : <style type="text/css"></style>
    Element style akan kita pelajari, saat memasuki materi pembelajaran CSS.
    
7. Element NOSCRIPT
    Element NOSCRIPT digunakan untuk menampilkan informasi ketika browser mematikan javascriptnya.
    Element NOSCRIPT ditulis seperti ini : <noscript>Silahkan aktifkan javascript, untuk mengakses situs ini</noscript>
    Karena ini sudah masuk hal teknis, maka element noscript hanya bisa kita praktekkan saat kita sudah memasuki materi pembuatan website dengan javascript didalamnya.

PERTEMUAN 9 : Comment dan UnComment

1. Comment
    Dalam HTML Comment digunakan untuk memberikan keterangan pada code yang kita tulis didalam HTML, sehingga code yang kita tulis hari ini ketika kita membukanya dikemudian hari, kita akan mudah memahaminya dengan ada comment tersebut.

    Code/Teks yang dicomment dalam HTML tidak akan diproses oleh web browser, sehingga ketika dijalankan code/teks tersebut tidak akan ditampilkan.
    Comment dibuat menggunakan :
        tag pembuka     : <!--
        tag penutup     : -->
    
    Contoh : 
        <h1>Element Untuk Judul H1</h1>
        <h2>Element Untuk Judul H2</h2>
        <!-- <h3>Element Untuk Judul H3</h3> -->
        <h4>Element Untuk Judul H4</h4>
        <!-- <h5>Element Untuk Judul H5</h5> -->
        <h6>Element Untuk Judul H6</h6>

    Untuk melakukan comment kita bisa melakukaan secara manual dengan cara menuliskan tag pembuka (<!--) dan penutup (-->) pada keterangan ataupun element yang ingin kita comment.

    Tapi kita juga bisa melakukannya lewat shortcut keys pada keyboard, caranya :
    - Blok/Tandai Code yang ingin kita jadikan comment
    - Tekan CTRL + / (Windows/Linux) atau Command + / (Mac OS)

2. UnComment
    Untuk UnComment atau menghapus comment kita bisa melakukannya secara manual dengan menghapus tag pembuka (<!--) dan penutup (-->) pada element comment yang ingin kita UnComment.

    Tapi kita juga bisa melakukannya lewat shortcut keys pada keyboard, caranya :
    - Blok/Tandai Code (yang sudah dicomment) yang ingin kita UnComment
    - Tetan CTRL + / (Windows/Linux) atau Command + /

PERTEMUAN 12 : Element Span

Element Span merupakan element inline (element yang tidak membentuk garis baru) yang digunakan untuk mengelompokkan beberapa element yang dimaksudkan untuk memudahkan dalam pemberian style (dengan CSS)
Contoh : 
    Jika kita menuliskan kode berikut
    <span>jalan 1</span>
    <span>jalan 2</span>

    Maka saat tampil di web browser teks "jalan 1" dan "jalan 2" akan sejaajar tanpa membuat baris baru, berbeda dengan saat kita menggunakan element paragraph.
    Itu artinya menggunakan element span tidak akan merubah struktur teks yang sudah kita buat.

Namun kegunaan span itu sendiri lebih ke mengelompokkan dan digunakan untuk mempermudah pemberian code CSS pada teks tertentu.
Contoh : 
    <p>Belajar Membuat <span>Website</span></p>
    Sehingga kita bisa memberikan code CSS khusus pada kata "Website" saja, seperti berikut ini :
        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Belajar HTML  - NIOMIC</title>
            <style>
                .web{
                color: orange;
                }
            </style>
        </head>
        <body>
            <p>Belajar Membuat <span class="web">Website</span></p>
        </body>
        </html>
    Yang mana ketika dijalankan, maka hanya kata "Website" saja yang akan berwarna orange.

PERTEMUAN 14 : Anchor Element

Anchor Element yang ditulis <a></a> merupakan salah satu tag terpenting dalam HTML. Anchor Element ini digunakan untuk membuat link ke halaman lain.
Contoh Penulisan : <a>Klik Disini</a>

1. Atribut href
    Untuk menginput lokasi tujuan pada element <a></a>, maka membutuhkan atribut href (hyperText references)
    - href dengan tujuan website lain :
        <a href="https://niomic.com">Klik Disini</a>
    - href dengan tujuan halaman sendiri di dalam web :
        Untuk mencoba menggunakan href dengan tujuan halaman yang berada di dalam 1 web, sekarang kita akan mencoba membuat 4 buah file html dan 2 folder, dengan susunan sebagai berikut :
        
        html
            blog
                contact
                    halaman3.html
                halaman2.html
            halaman1.html
            index.html
        
        - Buatlah file index.html dan halaman1.html dan folder dengan nama blog
        - Di dalam folder blog buatlah file halaman2.html dan folder dengan nama contact
        - Di dalam folder contact buatlah file dengan nama halaman3.html

        index.html : Pada halaman index.html, kita akan menuliskan code element anchor yang mengarah ke halaman1.html
            <a href="halaman1.html">Ke Halaman 1</a>
        
        halaman1.html : Pada halaman halaman1.html, kita akan menuliskan code element anchor yang mengarah ke halaman2.html
            <a href="blog/halaman2.html">Ke Halaman 2</a>
        
        halaman2.html : Pada halaman halaman2.html, kita akan menuliskan code element anchor yang mengarah ke halaman3.html
            <a href="contact/halaman3.html">Ke Halaman 3</a>

        halaman3.html : Pada halaman halaman3.html, kita akan menuliskan code element anchor yang mengarah Kembali ke index.html
            <a href="../../index.html">Ke Halaman Index</a>

3. Atribut Target
    Dengan menggunakan atribut target="_blank" kita bisa membuka tab baru untuk halaman web yang kita tuju
        <a href="https://niomic.com" target="_blank">Klik Halaman Baru</a>
    
    Kita bisa mengklik "Klik Halaman Baru" dan akan otomatis membuka tab baru dan akan menampilkan halaman web yang dituju sesuai value dari href

PERTEMUAN 15 : Mencoba Link Anchor Element Dengan Tujuan ID

Anchor element yang ditulis <a></a> merupakan salah satu tag terpenting dalam HTML.
Anchor element ini digunakan untuk membuat link ke halaman lain atau bisa juga ke halaman itu sendiri

Contoh penulisan : <a>Klik Disini</a>

- Atribut id
    Untuk mencoba menghubungkan sebuah link ke bagian halaman yang sama, kita membutuhkan sebuah atribut id yang mana atribut id berfungsi untuk memberi tanda pada sebuah element.

Untuk mencoba membuat contoh link ke halaman yang sama, maka kita akan membutuhkan sebuah halaman yang memiliki banyak konten.

1. Kita berikan id pada setiap element judul dalam sebuah paragraf
    <h3 id="page1">Judul Artikel 1</h3>
    <h3 id="page2">Judul Artikel 2</h3>
    <h3 id="page3">Judul Artikel 3</h3>

2. Kita tuliskan kode anchor element yang ketika di klik nanti akan mengarahkan ke setiap judul paragraf setiap halaman tersebut.
    <a href="#page1">Ke Artikel 1</a>
    <a href="#page2">Ke Artikel 2</a>
    <a href="#page3">Ke Artikel 3</a>

3. Dan untuk melihat hasilnya kita harus membuat sebuah paragraf yang panjang dalam halaman tersebut, agar terlihat perubahannya ketika kita menjalankan dan mencobanya nanti.

PERTEMUAN 18 : Formatting Element

1. Italic Element
    Digunakan untuk membuat efek miring pada huruf
    <i></i>
2. Bold Element
    Digunakan untuk membuat efek tebal pada huruf
    <b></b>
3. Underline Element
    Digunakan untuk memberi efek garis bawah pada teks
    <u></u>
4. Strikethrough Element
    Digunakan untuk memberi efek teks tercoret
    <s></s>
5. Superscript Element
    Digunakan untuk membuat karakter kecil diatas pada karakter yang ditandai (Misalnya untuk membuat operasi perpangkatan)
    <sup></sup>
6. Subscript Element
    Digunakan untuk membuat karakter kecil dibawah pada karakter yang ditandai
    <sub></sub>
7. Preformatted Element
    Digunakan untuk membuat teks menjadi tidak terformat
    <pre></pre>
