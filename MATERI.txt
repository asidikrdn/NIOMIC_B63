PERTEMUAN 1 : Apa itu HTML

HTML
HTML merupakan singkatan dari Hypertext Markup Language. Singkatan ini terdiri dari 3 komponen kata, yakni: Hypertext, Markup dan Language yaitu bahasa markup standar yang digunakan untuk membuat halaman web.
HyperText Markup Language (HTML) adalah bahasa inti dari hampir semua konten Web. Sebuah bahasa yang menggambarkan struktur dan isi semantik dari sebuah dokumen Web. Sama seperti yang dijelaskan dalam wikipedia tentang HTML : https://id.wikipedia.org/wiki/HTML
Saat menjelaskan apa itu HTML, kita tidak bisa melepaskan dengan CSS (Cascading Style Sheets) dan JS (Javascript).  Untuk membuat sebuah website kita tidak bisa memisahkan antara ketiga hal tersebut yaitu HTML, CSS dan JS.

Untuk mempermudah dalam memahami apa itu HTML, mari kita buat sebuah perumpamaan seperti membangun sebuah rumah ?
    HTML    : Diibaratkan seperti membangun sebuah rumah maka HTML itu adalah kerangkanya saja, hanya kerangka sebuah rumah
    CSS     : Akan memiliki fungsi untuk mengatur warna, ketinggian, panjang, atau lebar dari sebuah pintu, jendela, dinding dll. Intinya CSS memiliki fungsi untuk mengatur semua warna, bentuk ataupun ukuran dari panjang lebarnya sebuah rumah.
    JS      : Akan memiliki fungsi untuk mengatur pintu rumah dibuka kedalam / keluar, mengatur saluran air, mengatur instalasi listrik seperti Ketika di Tekan ON maka lampu kamar menyala.

PERTEMUAN 3 : Struktur HTML

Kita akan memahami istilah-istilah dasar pada HTML yaitu :
    1. Element HTML
        Jika kita bilang Element, maka yang dimaksud adalah satu bagian utuh yang terdiri dari tag, atribut, dan seluruh teks yang berada diantara tag pembuka dan penutup.
        namun untuk mempermudah memahami Element, berikut adalah beberapa contoh dari element :
        - Element Meta      : <meta charset="utf-8">
        - Element HTML      : <html></html>
        - Element Head      : <head></head>
        - Element title     : <title></title>
        - Element body      : <body></body>
    
    2. Tag (Pembuka dan Penutup) dan Self Closing Tag 
        Tag adalah bagian dari sebuah element, tag sendiri ada yang disebut : tag pembuka, tag penutup, dan self Closing tag.
        Kita akan coba memahami ketiga hal tersebut. 
        - Tag Pembuka
            Pada element body, yang disebut tag pembuka adalah      : <body>
            Pada element tittle, yang disebut tag pembuka adalah    : <tittle>
        - Tag Penutup
            Pada element body, yang disebut tag penutup adalah      : </body>
            Pada element tittle, yang disebut tag penutup adalah      : </tittle>
        - Self Closing Tag
            Self Closing Tag adalah element dalam html yang tidak memiliki pasangan tag pembuka dan penutup.
            Contoh  : <meta charset="utf-8">

            Ada banyak element yang sifatnya self closing tag seperti :
            <area />   <base />   <br />   <col />   <command />   <embed />   <hr />   <img />   <input />
            <keygen />   <link />   <menuitem />   <meta />   <param />   <source />   <track />   <wbr />

            Penulisan "/" pada self closing tag dalam HTML 5 bersifat optional, dan lebih baik ditinggalkan. 
            Sehingga nanti ketika kita belajar menuliskan element yang sifatnya self closing tag, kita cukup menuliskannya seperti ini :
            <meta>, <img>, <hr>, <br>, dan seterusnya.

            Namun sekarang kita tidak akan membahas fungsi dari element-element tersebut, kita saat ini hanya perlu memahami istilah-istilah yang berada di dalam element HTML.

    3. Attributes
        Attribute adalah informasi tambahan yang ditulis pada tag pembuka. 
        Fungsi dari atribut ini bermacam-macam tergantung nilai dan pada tag mana ia ditempatkan.
        Untuk memahami atribut maka kita akan menuliskan contoh element yang didalamnya memiliki atribut :
        
        <a id="info" href="https://niomic.com">Info Pembelajaran HTML</a>

        Ket :
        - Element a         : <a id="info" href="https://niomic.com">Info Pembelajaran HTML</a>
        - Tag Pembuka       : <a id="info" href="https://niomic.com">
        - Tag Penutup       : </a>
        - Attribute         : id="info"
                              href="https://niomic.com"
            Nama Attribute  : id dan href
            Value/Nilai     : "info" dan "https://niomic.com"

    4. DOCTYPE / DTD (Document Type Declaration)
        DOCTYPE atau Document Type Declaration (DTD) adalah sebuah keterangan yang ditulis untuk memberitahu web browser tentang aturan penulisan dari dokumen yang sedang ditampilkan. 
        Doctype ditulis pada baris pertama halaman HTML.
        Pada HTML versi 4.01, DOCTYPE ditulis seperti ini :
        <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
        "https://www.w3.org/TR/html14/strict.dtd">

        Pada HTML 5 Penulisan doctype jauh lebih sederhana yaitu : <!DOCTYPE html>
    
    5. Body Element
        Di dalam element body inilah tempat kita akan menuliskan code-code HTML yang akan kita tampilkan pada web browser

PERTEMUAN 6 : HEAD Element

Element yang sering digunakan dan diletakkan di dalam element head :
1. Element Meta
    Tag <meta> berfungsi untuk memberi informasi mengenai halaman HTML yang saat ini sedang ditampilkan
    - Meta charset  : berfungsi memberi instruksi kepada web browser tentang bagaimana cara menerjemahkan karakter-karakter yang ada di dalam halaman HTML.
    Pada HTML 4 meta charset ditulis seperti ini : <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    Namun pada HTML 4 penulisannya jauh lebih ringkas : <meta charset="UTF-8">
    Ketika kita menuliskan struktur HTML dan didalamnya kita letakkan meta caharset ataupun tidak, kita tidak bisa melihat perbedaannya karena meta charset ini sendiri berhubungan bagaimana web browser memproses pada karakter-karakter yang berada di dalam HTML.

    - Meta Author, Keywords dan Description
        Ketika kita menuliskan Meta Author, Keywords, dan Description maka ketiga meta tersebut berhubungan dengan SEO (Search Engine Optimization) yaitu digunakan untuk mempermudah mesin pencari seperti google untuk mendapatkan informasi dari website yang kita buat.
        Sehingga ketika kita memasang ketiga meta tersebut di dalam struktur HTML kita, kita tidak bisa melihat perbedaannya karena itu berhubungan dengan web browser memberikan informasi kepada 'robot' / mesin pencari.
        Contoh Penulisan : 
        * <meta name="author" content="Makanan Nusantara">
        * <meta name="keywords" content="Rendang, Bubur Menado, Gulai Nangka Muda, Lontong Tahu, Soto Makasan">
        * <meta name="description" content="Situs Tentang Makanan Nusantara">

    - Meta Refresh
        Meta refresh digunakan untuk melakukan web refresh / reload pada halaman website kita, Contoh : jika kita ingin web refresh / reload website kita setiap 60 detik sekali, maka kita cukup menuliskan code meta refresh.
        <meta http-equiv="refresh" content="60">
        
    -Meta tag Viewport
        Meta Viewport yang ditulis seperti ini : <meta name="viewport" content="width=device-width, initial-scale=1">
        Digunakan pada web dengan desain responsive (Responsive Web Design). Secara garis besar, mea tag viewport ditambahkan agar desain responsive dapat tampil sempurna di dalam layar berukuran kecil (seperti tablet atau smartphone).
        Meta tag viewport ini nanti akan kita praktekkan saat belajar dengan web responsive di materi selanjutnya.
        
    NOTE : Untuk Meta tag X-UA-Compatible dan Meta tag Robots, saat ini bekum diterapkan karena hal ini sudah masuk ke tingkat teknis, jika nanti sudah mampu membuat sebuah website dengan HTML silahkan kembali untuk mempelajari kedua meta tag tersebut.

    - Meta tag X-UA-Compatible
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        Digunakan untuk menangani permasalahan terkait web browser Internet Explorer. IE memiliki mode compatibility view yang aktif ketika halaman web diakses dari komputer lokal atau diakses dari dalam Intranet.
        Fitur ini membuat perlakuan berbeda dalam menangani halaman web (mirip quirks mode).
        Meta tag diatas akan memaksa IE untuk tidak masuk ke mode compatibility view.

    - Meta tag Robots
        <meta name="robots" content="index, follow"> atau <meta name="robots" content="noindex, nofollow">
        Digunakan untuk memberi pesan kepada search engine bagaimana halaman web akan diproses, jika nilai atribut content adalah "index, follow", maka mesin pencari akan mengindex halaman tersebut (dimasukkan kedalam hasil pencarian).
        Namun jika nilai atribut content adalah "noindex, nofollow", maka isi web tidak akan dimasukkan ke hasil pencarian (mungkin halaman web bersifat rahasia dan anda tidak ingin seseorang mengaksesnya dari google).
    
2. Element Title
    Element title berfungsi untuk menampilkan judul pada halaman website yang kita buat. 
    Contoh : 
    <title>Belajar HTML - NIOMIC</title>

    Jika dijalankan maka tab pada browser saat membuka file HTML kita akan tampil tulisan sesuaai dengan isi dari element title.

3. Element Link
    Element link akan sering kita gunakan ketika kita berhubungan dengan file lain, contoh ketika kita ingin menghubungkan HTML kita dengan file css,
    codenya : <link href="style.css" rel="stylesheet">
    Kita akan membahas hal tersebut nanti saat kita memasuki pembelajaran CSS.

    Contoh menggunakan element link untuk membuat Favicon (icon yang ditampilan pada tab browser)
    <link rel="icon" href="http://www.iconarchive.com/download/i97515/sicons/basic-round-social/facebook.ico" type="image/x-icon">
    - Gambar dari internet
        Pada contoh diatas, kita menampilkan Favicon dari internet yaitu melalui link icon tersebut
    - Gambar dari local
        Jika kita meletakkan gambar dalam folder tempat kita menuliskan code HTMLnya
        <link rel="icon" href="namaicon.png" type="image/png">
        Jika gambar kita letakkan di dalam folder lagi, misal folder img, maka seperti berikut
        <link rel="icon" href="img/namaicon.jpg" type="image/jpg">
        
    NOTE :
    pada atribut type="image/png" kita bisa menggantinya sesuai dengan gambar yang kita masukkan, jika gambar yang dimasukkan adalah jpg maka atribut type menjadi type="image/jpg",
    dan jika gambarnya berformat .ico maka menjadi type="image/x-icon".
    
4. Element Base
    Element base bersifat global, dan digunakan untuk memasukkan sebuah link secara global.
    Element base ini akan kita coba praktekkan saat kita belajar materi HTML selanjutnya, karena element base berhubungan dengan element lain, agar ketika dipraktekkan menjadi lebih paham.
    Element base ditulis seperti ini : <base href="/media/images" target="_blank">
    
5. Element Script
    Element Script digunakan untuk menginput Script kedalam halaman HTML, element script sering digunakan untuk menuliskan code Javascript, baik yang dituliskan secara internal ataupun menghubungkan secara eksternal.
    Code ditulis seperti ini : <script type="text/javascript"></script>
    Element script akan kita praktekkan nanti saat kita mulai belajar javascript.

6. Element Style
    Element Style digunakan untuk menginput kode style ke dalam halaman HTML.
    Element Style ditulis seperti ini : <style type="text/css"></style>
    Element style akan kita pelajari, saat memasuki materi pembelajaran CSS.
    
7. Element NOSCRIPT
    Element NOSCRIPT digunakan untuk menampilkan informasi ketika browser mematikan javascriptnya.
    Element NOSCRIPT ditulis seperti ini : <noscript>Silahkan aktifkan javascript, untuk mengakses situs ini</noscript>
    Karena ini sudah masuk hal teknis, maka element noscript hanya bisa kita praktekkan saat kita sudah memasuki materi pembuatan website dengan javascript didalamnya.

PERTEMUAN 9 : Comment dan UnComment

1. Comment
    Dalam HTML Comment digunakan untuk memberikan keterangan pada code yang kita tulis didalam HTML, sehingga code yang kita tulis hari ini ketika kita membukanya dikemudian hari, kita akan mudah memahaminya dengan ada comment tersebut.

    Code/Teks yang dicomment dalam HTML tidak akan diproses oleh web browser, sehingga ketika dijalankan code/teks tersebut tidak akan ditampilkan.
    Comment dibuat menggunakan :
        tag pembuka     : <!--
        tag penutup     : -->
    
    Contoh : 
        <h1>Element Untuk Judul H1</h1>
        <h2>Element Untuk Judul H2</h2>
        <!-- <h3>Element Untuk Judul H3</h3> -->
        <h4>Element Untuk Judul H4</h4>
        <!-- <h5>Element Untuk Judul H5</h5> -->
        <h6>Element Untuk Judul H6</h6>

    Untuk melakukan comment kita bisa melakukaan secara manual dengan cara menuliskan tag pembuka (<!--) dan penutup (-->) pada keterangan ataupun element yang ingin kita comment.

    Tapi kita juga bisa melakukannya lewat shortcut keys pada keyboard, caranya :
    - Blok/Tandai Code yang ingin kita jadikan comment
    - Tekan CTRL + / (Windows/Linux) atau Command + / (Mac OS)

2. UnComment
    Untuk UnComment atau menghapus comment kita bisa melakukannya secara manual dengan menghapus tag pembuka (<!--) dan penutup (-->) pada element comment yang ingin kita UnComment.

    Tapi kita juga bisa melakukannya lewat shortcut keys pada keyboard, caranya :
    - Blok/Tandai Code (yang sudah dicomment) yang ingin kita UnComment
    - Tetan CTRL + / (Windows/Linux) atau Command + /

PERTEMUAN 12 : Element Span

Element Span merupakan element inline (element yang tidak membentuk garis baru) yang digunakan untuk mengelompokkan beberapa element yang dimaksudkan untuk memudahkan dalam pemberian style (dengan CSS)
Contoh : 
    Jika kita menuliskan kode berikut
    <span>jalan 1</span>
    <span>jalan 2</span>

    Maka saat tampil di web browser teks "jalan 1" dan "jalan 2" akan sejaajar tanpa membuat baris baru, berbeda dengan saat kita menggunakan element paragraph.
    Itu artinya menggunakan element span tidak akan merubah struktur teks yang sudah kita buat.

Namun kegunaan span itu sendiri lebih ke mengelompokkan dan digunakan untuk mempermudah pemberian code CSS pada teks tertentu.
Contoh : 
    <p>Belajar Membuat <span>Website</span></p>
    Sehingga kita bisa memberikan code CSS khusus pada kata "Website" saja, seperti berikut ini :
        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Belajar HTML  - NIOMIC</title>
            <style>
                .web{
                color: orange;
                }
            </style>
        </head>
        <body>
            <p>Belajar Membuat <span class="web">Website</span></p>
        </body>
        </html>
    Yang mana ketika dijalankan, maka hanya kata "Website" saja yang akan berwarna orange.

PERTEMUAN 14 : Anchor Element

Anchor Element yang ditulis <a></a> merupakan salah satu tag terpenting dalam HTML. Anchor Element ini digunakan untuk membuat link ke halaman lain.
Contoh Penulisan : <a>Klik Disini</a>

1. Atribut href
    Untuk menginput lokasi tujuan pada element <a></a>, maka membutuhkan atribut href (hyperText references)
    - href dengan tujuan website lain :
        <a href="https://niomic.com">Klik Disini</a>
    - href dengan tujuan halaman sendiri di dalam web :
        Untuk mencoba menggunakan href dengan tujuan halaman yang berada di dalam 1 web, sekarang kita akan mencoba membuat 4 buah file html dan 2 folder, dengan susunan sebagai berikut :
        
        html
            blog
                contact
                    halaman3.html
                halaman2.html
            halaman1.html
            index.html
        
        - Buatlah file index.html dan halaman1.html dan folder dengan nama blog
        - Di dalam folder blog buatlah file halaman2.html dan folder dengan nama contact
        - Di dalam folder contact buatlah file dengan nama halaman3.html

        index.html : Pada halaman index.html, kita akan menuliskan code element anchor yang mengarah ke halaman1.html
            <a href="halaman1.html">Ke Halaman 1</a>
        
        halaman1.html : Pada halaman halaman1.html, kita akan menuliskan code element anchor yang mengarah ke halaman2.html
            <a href="blog/halaman2.html">Ke Halaman 2</a>
        
        halaman2.html : Pada halaman halaman2.html, kita akan menuliskan code element anchor yang mengarah ke halaman3.html
            <a href="contact/halaman3.html">Ke Halaman 3</a>

        halaman3.html : Pada halaman halaman3.html, kita akan menuliskan code element anchor yang mengarah Kembali ke index.html
            <a href="../../index.html">Ke Halaman Index</a>

3. Atribut Target
    Dengan menggunakan atribut target="_blank" kita bisa membuka tab baru untuk halaman web yang kita tuju
        <a href="https://niomic.com" target="_blank">Klik Halaman Baru</a>
    
    Kita bisa mengklik "Klik Halaman Baru" dan akan otomatis membuka tab baru dan akan menampilkan halaman web yang dituju sesuai value dari href

PERTEMUAN 15 : Mencoba Link Anchor Element Dengan Tujuan ID

Anchor element yang ditulis <a></a> merupakan salah satu tag terpenting dalam HTML.
Anchor element ini digunakan untuk membuat link ke halaman lain atau bisa juga ke halaman itu sendiri

Contoh penulisan : <a>Klik Disini</a>

- Atribut id
    Untuk mencoba menghubungkan sebuah link ke bagian halaman yang sama, kita membutuhkan sebuah atribut id yang mana atribut id berfungsi untuk memberi tanda pada sebuah element.

Untuk mencoba membuat contoh link ke halaman yang sama, maka kita akan membutuhkan sebuah halaman yang memiliki banyak konten.

1. Kita berikan id pada setiap element judul dalam sebuah paragraf
    <h3 id="page1">Judul Artikel 1</h3>
    <h3 id="page2">Judul Artikel 2</h3>
    <h3 id="page3">Judul Artikel 3</h3>

2. Kita tuliskan kode anchor element yang ketika di klik nanti akan mengarahkan ke setiap judul paragraf setiap halaman tersebut.
    <a href="#page1">Ke Artikel 1</a>
    <a href="#page2">Ke Artikel 2</a>
    <a href="#page3">Ke Artikel 3</a>

3. Dan untuk melihat hasilnya kita harus membuat sebuah paragraf yang panjang dalam halaman tersebut, agar terlihat perubahannya ketika kita menjalankan dan mencobanya nanti.

PERTEMUAN 21 : List Element

1. Ordered List
    Digunakan untuk membuat list yang berurutan.
    Contoh :
    <ol>
        <li>Nasi Kucing</li>
        <li>Kue Dolar</li>
        <li>Roti Ketawa</li>
        <li>Kue Satu</li>
        <li>Kue Kentut</li>
    </ol>

    Atribut type
        Secara default element ordered list akan menampilkan urutan berupa angka dari 1 sampai panjang element <li>.
        Jika ingin mengganti bukan dengan angka, contoh berupa huruf "a"/"A" atau huruf romawi "i"/"I", kita bisa melakukannya dengan cara memberikan atribut type pada element <ol>.
        Contoh :
        <ol type="a">
        atau
        <ol type="A">
        atau
        <ol type="i">
        atau
        <ol type="I">
        atau
        <ol type="1">
    
    Atribut start
        Atribut start digunakan untuk mengSET dari mana nomor urut dimulainya
        Contoh :
        <ol type="a" start="4">

        maka list yang ditampilkan akan dimulai dari huruf "d"
    
    Membuat Nested List
        Kita juga bisa membuat nested list, yaitu list yang didalamnya juga memiliki list
        Contoh :
        <ol type="1" start="4">
            <li>Nasi Kucing</li>
                <ol type="a">
                    <li>Cabang Yogya</li>

PERTEMUAN 24 : Image Element

Element Image digunakan untuk menampilkan gambar ke dalam web browser.
Element Image bersifat self closing dan ditulis : <img>

- Atribut Src
    Untuk menampilkan gambar, kita dapat menambahkan atribut src yang berfungsi mengambil gambar baik dari internet maupun local.
    Contoh :
    <img src="https://cdn-icons-png.flaticon.com/512/1312/1312139.png">     (Gambar Online)
    <img src="img/contoh-gambar.png">                                       (Gambar Lokal)

- Atribut Alt
    Penulisan atribut alt ini sangat penting di dalam element image tersebut. Atribut alt ini memiliki fungsi untuk menampilkan teks ketika gambar yang kita masukkan gagal terload/terbuka di browser.
    Contoh :
    Gambar yang berada di lokal kita bernama "contoh-gambar.png" dan akan saya coba salahkan penulisannya dengan menuliskan "contoh-gambar1.png", sehingga gambar tersebut gagal terload
    <img src="img/contoh-gambar.png" alt="gambar gorengan">

- Atribut Title
    Atribut ini berfungsi untuk memberikan keterangan pada saat cursor mouse kita berada/diarahkan pada gambar tersebut.
    Contoh :
    <img src="http://glutenfreedelivers.com/template/images/white/glutenfree4.jpg" alt="mie instan" title="Indomie">

- Atribut Height & With
    Element Image memiliki atribut yang berfungsi untuk mengatur lebar dan tinggi dari image tersebut, kita bisa melakukannya dengan menambahkan atribut width dan height dalam satuan pixel/persen.
    Contoh :
    Pixel   : <img src="img/contoh-gambar.png" alt="gambar gorengan" width="200" height="200">
    Persen  : <img src="img/contoh-gambar.png" alt="gambar gorengan" width="50%" height="40%">

- Atribut Border
    Atribut Border berfungsi untuk memberikan garis tepi/bingkai. Value dari atribut border satuannya adalah pixel.
    Contoh :
    <img src="img/contoh-gambar.png" alt="gambar gorengan" width="50%" height="40%" border="4">

- Atribut Align
    Atribut Align berfungsi untuk mengatur posisi dari gambar terhadap teks di sekitarnya.
    Ada 4 value dalam atribut align ini : bottom, left, middle, right, top.
    Untuk mencoba mengaplikasikan atribut tersebut kita membutuhkan teks, agar nanti terlihat perbedaannya.
    Contoh : 
    <img src="img/contoh-gambar.png" alt="gambar gorengan" width="10%" height="10%" border="4" align="right">

PERTEMUAN 27 : Menerapkan Meta Tag Viewport

Meta tag viewport ditambahkan agar desain responsive dapat tampil sempurna di dalam layar berukuran kecil (seperti tablet atau smartphone). Sekarang kita akan coba mempraktekkannya.

Contoh, Meta tag viewport ditulis seperti ini : 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Dan untuk menguji meta tag viewport kita memasukkan sebuah image, dan sebuah teks paragraf seperti berikut :
<!DOCTYPE html>
<html>
 <head>
   <meta charset="utf-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Belajar HTML  - NIOMIC</title>
 </head>
 <body>
     <img src="img/contoh-gambar.png" alt="gambar gorengan" width="350" height="350">
     <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.</p>
 </body>
</html>

Untuk melihat perbedaannya sekarang kita praktekkan bertahap :
1. Silahkan comment element meta tag viewport
2. Kemudian jalankan di browser, silahkan di inspect element dan kemudian dirubah browsernya ke versi mobile, maka akan terlihat hasilnya. Ketika kita menonaktifkan meta tag viewport kita bisa melihat bahwa hasilnya tidak terlihat responsive untuk versi mobile.
3. Sekarang coba kita aktifkan kembali meta tag viewport
4. Kemudian buka kembali browser yang sudah dalam posisi versi mobile, Dan sekarang kita lihat hasilnya. Kita bisa melihat hasilnya bahwa gambar yang ditampilkan akan jauh lebih bagus ketika kita menggunakan meta tag viewport untuk membuat tampilan tersebut menjadi responsive.

PERTEMUAN 29 : Iframe Element

Iframe digunakan untuk memasukkan halaman dari luar ke dalam website kita, bisa dalam bentuk file pdf, youtube, website lain, ataupun maps.
Penulisan :
<iframe src=""></iframe>

1. Website
    Dengan iframe kita bisa memasukkan website lain ke dalam halaman kita.
    
    Contoh :
    <iframe src="https://niomic.com" width="600" height="600"></iframe>

    Di dalam iframe kita dapat mengatur atribut width dan height nya sesuai kebutuhan kita. 
    
    Note : Tidak semua website bisa dimasukkan ke dalam iframe, itu tergantung kebijakan pemilik websitenya apakah membolehkan atau tidak.

2. File PDF
    Kita juga bisa memasukkan file pdf ke dalam iframe, sebagai contoh kita akan mencoba memasukkan file pdf ini : https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf

    Contoh :
    <iframe src="https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" width="600" height="600"></iframe>

3. Youtube
    Youtube sudah menyediakaan video-videonya untuk dimasukkan ke halaman website kita, dan kita hanya cukup untuk mengambil link iframe yang sudah disediakan oleh youtube. Caranya :
    - Buka video youtube yang disematkan di halaman website kita
    - Klik Bagikan
    - Kemudian Klik Sematkan
    - Kemudian akan muncul link iframenya, silahkan copy dan masukkan ke dalam halaman website kita

    Contoh :
    <iframe width="560" height="315" src="https://www.youtube.com/embed/ZoUFPBrSO1o" frameborder="0" allow="autoplay; encrypted-media" allowfullscreen></iframe>

4. Maps
    Sama seperti youtube, google maps juga bisa dimasukkan ke dalam website kita, dan google maps juga sudah menyediakan link iframe untuk dimasukkan ke dalam website kita. Caranya :
    - Bukalah maps google, dan pilih lokasi
    - Klik menu Bagikan dan kemudian akan muncul pilihan
    - Silahkan klik Sematkan Peta, kemudian kamu bisa mengcopy link iframe yang tersedia
    
    Contoh :
    <iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d1016369.0810230912!2d105.72958496230609!3d-5.685409760804671!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x2e69f5d2e764b12d%3A0x3d2ad6e1e0e9bcc8!2sMonumen+Nasional!5e0!3m2!1sid!2sid!4v1539336172375" width="600" height="450" frameborder="0" style="border:0" allowfullscreen></iframe>

PERTEMUAN 31 : Audio Element

Dengan audio element, kita dapat menjalankan file audio di dalam website kita. 
Penulisaan audio element adalah : 
<audio></audio>

- Atribut src
    Untuk memasukkan file audio kita membutuhkan atribut src
    Contoh :
    <audio src="sample.mp3"></audio>

- Atribut controls
    Kita membutuhkan atribut ini agar audio dapat dijalankan di website kita. Sesuai namanya atribut ini berfungsi untuk memberikan control pada audio player kita.
    Contoh :
    <audio src="sample.mp3" controls></audio>

- Atribut muted
    Digunakan untuk mematikan volume audio
    Contoh :
    <audio src="sample.mp3" controls muted></audio>

- Atribut loop
    Atribut ini digunakan untuk memutar kembali audio yang sudah selesai diputar.
    Contoh :
    <audio src="sample.mp3" controls loop></audio>

- Atribut preload
    Atribut ini berfungsi untuk mengatur bagaimana browser untuk men-buffer / mendownload audio sebelum diplay.
    Ada 3 value pada atribut ini :
    - auto  : artinya saat web diload, audio akan otomatis di buffer/download sehingga saat diplay tidak buffer lagi
    - none  : artinya saat web diload, audio tidak akan otomatis di buffer/download hingga user menekan tombol play
    - meta  : artinya saat web diload, audio akan di buffer/download sebagian dan sebagian yang lain saat diplay

- Memuat 2 format audio dalam 1 element audio
    Tentunya akan ada beberapa browser yang tidak support dengan sebuah audio. Untuk mengatasi hal tersebut kita bisa memasukkan beberapa jenis audio ke dalam element audio, yang nantinya akan berfungsi memberikan alternatif jika salah satu audio tidak berfungsi akan memutar format audio yang lain

    Contoh :
    <audio controls>
        <source src="sample.mp3" type="audio/mp3">
        <source src="sample.ogg" type="audio/ogg">
    </audio>

- Atribut autoplay
    Atribut ini berfungsi untuk membuat audio kita langsung terplay ketika halaman website diakses. Namun kadang-kadang terjadi kendala pada autoplay di beberapa browser.
    Contoh :
    <audio src="sample.mp3" controls autoplay></audio>

    Contoh:
    <video controls>
        <source src="sample.mp4" type="video/mp4">

PERTEMUAN 36 : Form Element

Element Form akan sangat sering kita gunakan ketika kita membangun sebuah website, terutama untuk membuat sesuatu yang berhubungan dengan input data, form digunakan sebagai container / pembungkus dari element seperti text box, text area, radio button, dll.
Contoh :
<form></form>

- Atribut Action
    Berfungsi untuk merujuk pada file untuk tujuan dari eksekusi pengiriman data
    Contoh :
    <form Action=""></form>

- Atribut Method
    Berfungsi memberitahu browser tentang proses pengiriman sebuah data. Value dari method ini bisa berupa "get" / "post".
    Contoh :
    <form action="" method="get"></form>
    <form action="" method="post"></form>

- Atribut Target
    Atribut target disini sama fungsinya dengan atribut target pada element anchor, dengan menggunakan atribut tersebut ketika form dieksekusi akan membuka tab baru (jika value diisi dengan _blank)
    Contoh :
    <form action="" method="post" target="_blank"></form>

- Atribut Novalidate
    Berfungsi untuk menonaktifkan validasi bawaan dari element yang berada di dalam element form.
    Contoh :
    <form action="" method="" Novalidate></form>

PERTEMUAN 37 : Input Element Type Text (Text Box)

Element Input merupakan element yang banyak digunakan ketika kita membangun sebuah website.
Contoh :
<input>

- Atribut Type (Text)
    Atribut Type berfungsi mendifinisikan jenis input pada element input, atribut type pada element input memiliki value / nilai yang berbeda-beda, salah satunya yang akan dibahas dipertemuan ini adalah "text".
    Contoh :
    <input type="text">

- Atribut Value
    Atribut Value berfungsi untuk memberikan nilai secara langsung pada element input tersebut.
    Contoh :
    <input type="text" value="Jakarta">

- Atribut Size
    Berfungsi untuk memnberikan ukuran element input berdasarkan jumlah karakter
    Contoh :
    <input type="text" Size="7">
    <br>
    <input type="text" size="12">

- Atribut ReadOnly dan Disabled
    Atribut ini berfungsi untuk membuat sebuah element input tidak dapat dirubah nilainya. Perbedaannya, ketika menggunakan disabled maka element tersebut benar-benar dinonaktifkan. Sedangkan readonly secara tampilan akan tetap aktif, hanya saja tidak dapat dirubah nilainya. 
    Contoh :
    <input type="text" value="Jakarta" disabled>
    <br>
    <input type="text" value="Jakarta" readonly>

- Atribut Maxlength
    Berfungsi untuk membatasi jumlah karakter yang diinput. Perbedaannya dengan atribut size adalah, atribut size membatasi secara tampilan, sedangkan Maxlength membatasi valuenya.
    Contoh :
    <input type="text" Maxlength="7">

- Atribut Autofocus
    Berfungsi untuk meletakkan kursor secara otomatis ketika halaman web diakses
    Contoh :
    <input type="text" autofocus>

- Atribut Placeholder
    Berfungsi untuk memberikan text pada element input yang berupa informasi sementara (bukan value). Biasanya digunakan untuk memberikan panduan terkait apa yang harus diisi dalam element tersebut.
    Contoh :
    <input type="text" placeholder="Nama Ibu Kota">

- Atribut Required
    Berfungsi untuk menandai bahwa element input tersebut wajib diisi.
    Contoh :
    <input type="text" Required>
    
    Untuk dapat melihat efek dari atribut Required, kita perlu melengkapi element input dengan atribut submit, dan diletakkan didalam element form.
    Contoh :
    <form>
        <input type="text" required>
        <input type="submit" value="KIRIM">
    </form>

    Jika ingin merubah pesan ketika element tersebut dikosongkan, gunakan atribut oninvalid.
    Contoh :
    <form>
        <input type="text" required oninvalid="this.setCustomValidity('Mohon diisi')">
        <input type="submit" value="KIRIM">
    </form>

- Atribut Name
    Berfungsi sebagai penanda sebuah element input di dalam form. Biasanya digunakan untuk pengenal pada bahasa lain, misal Javascript atau PHP.
    Contoh :
    <form>
        <input type="text" name="namadepan" value="">
        <input type="text" name="namabelakang" value="">
        <input type="text" name="kota" value="">
    </form>

PERTEMUAN 49 : Input Element Type Number (Min, Max, dan Step)

Element Input merupakan element yang banyak digunakan ketika kita membangun sebuah website.
Contoh :
<input>

- Atribut Type Number
    Atribut Type berfungsi mendifinisikan jenis input pada element input, atribut type pada element input memiliki value / nilai yang berbeda-beda, salah satunya yang akan dibahas dipertemuan ini adalah "number". Element input dengan atribut type number digunakan untuk input khusus angka.
    Contoh :
    <input type="number">

- Atribut Min dan Max
    Atribut ini digunakan untuk membatasi angka minimum dan maksimum yang boleh dimasukkan kedalam element input type number.
    Contoh :
    <input type="number" min="4" max="10">
    Dengan kode diatas, maka angka yang bisa dimasukkan adalah range dari 4 sampai 10.
    Selain pada type number, atribut min dan max juga dapat dikombinasikan dengan element input type lainnya, yaitu range, date, datetime-local, month, time, and week.

- Atribut Step
    Atribut ini digunakan untuk memberikan kelipatan ketika mengklik tombol keatas atau kebawah yang ada di kolom inputan.
    Contoh :
    <input type="number" min="4" max="10" step="2">
    Dengan kode diatas, maka ketika inputan awalnya adalah 4, begitu kita menekan tombol keatas maka inputan bertambah 2 menjadi 6.
    Selain pada type number, atribut step juga dapat dikombinasikan dengan element input type lainnya, yaitu range, date, datetime-local, month, time, and week.

PERTEMUAN 56 : Input Element Type Checkbox dan Radio

Element Input merupakan element yang banyak digunakan ketika kita membangun sebuah website.
Contoh :
<input>

- Atribut Type Checkbox
    Atribut Type berfungsi mendifinisikan jenis input pada element input, atribut type pada element input memiliki value / nilai yang berbeda-beda, salah satunya yang akan dibahas dipertemuan ini adalah "checkbox". Element input dengan atribut type checkbox digunakan untuk mendefinisikan kotak centang. Kotak centang ditampilkan sebagai kotak persegi yang dapat dicentang saat diaktifkan. Kotak centang digunakan untuk membiarkan pengguna memilih satu atau lebih opsi dari sejumlah pilihan terbatas.
    Contoh :
    <input type="checkbox">

- Atribut Type Radio
    Atribut Type berfungsi mendifinisikan jenis input pada element input, atribut type pada element input memiliki value / nilai yang berbeda-beda, salah satunya yang akan dibahas dipertemuan ini adalah "radio". Element input dengan atribut type radio digunakan untuk mendefinisikan tombol radio. Tombol radio biasanya disajikan dalam grup radio (kumpulan tombol radio yang menjelaskan serangkaian opsi terkait). Hanya satu tombol radio dalam grup yang dapat dipilih secara bersamaan.
    Contoh :
    <input type="radio">

    Catatan: Grup radio harus memiliki nama yang sama (nilai atribut nama) untuk diperlakukan sebagai grup. Setelah grup radio dibuat, memilih tombol radio apa pun di grup itu secara otomatis membatalkan pilihan tombol radio lain yang dipilih di grup yang sama. Anda dapat memiliki grup radio sebanyak mungkin di halaman yang Anda inginkan, selama setiap grup memiliki namanya sendiri.

    Catatan: Atribut value mendefinisikan nilai unik yang terkait dengan setiap tombol radio. Nilai tidak ditampilkan kepada pengguna, tetapi merupakan nilai yang dikirim ke server pada "kirim" untuk mengidentifikasi tombol radio mana yang dipilih.

- Atribut Checked
    Atribut ini berfungsi untuk menentukan element input type checkbox dan radio sebagai element yang tercentang (terpilih) sejak saat halaman web dimuat.
    Contoh :
    <input type="checkbox" name="makanan" checked>
    <input type="radio" name="minuman" checked>

PERTEMUAN 57 : Label Element

    Element Label digunakan untuk mendefinisikan label. Penggunaan element label tidak dapat langsung terlihat. Penggunaan label baru akan terasa berfungsi ketika kita mencoba mengklik kalimat/teks yang ada di dalam element label yang sudah direferensikan kepada element input tertentu maka kursor akan otomatis fokus pada element input tersebut. Hal ini berguna untuk memperluas area klik.

    Penulisan element label ada 2 cara, cara pertama ialah dengan memasukkan element input yang kita referensikan di dalam element label.
    Contoh :
    <label>Nasi Goreng : <input type="checkbox" name="makanan"></label>

    Cara kedua ialah menggunakan atribut for dan id. Atribut for dari <label> harus sama dengan atribut id dari elemen terkait untuk mengikatnya bersama.
    Contoh :
    <label for="NasiGoreng">Nasi Goreng</label>
    <input type="checkbox" name="makanan" id="NasiGoreng">

PERTEMUAN 59 : Textarea Element

Element textarea merupakan sebuah element yang digunakan untuk mendefinisikan kontrol input teks multi-baris. Element ini hampir sama dengan element input type text, bedanya pada textarea ukuran teks yang bisa dimuat jauh lebih banyak.
Elemen <textarea> sering digunakan dalam formulir, untuk mengumpulkan masukan pengguna seperti komentar atau ulasan.
Contoh :
<textarea></textarea>

- Atribut Rows dan Cols
    Kedua atribut ini berfungsi untuk mengatur jumlah baris (row) dan kolom (col) pada textarea, simpelnya kedua atribut ini digunakan untuk mengatur panjang dan lebar dari textarea.
    Contoh :
    <textarea rows="5" cols="20"></textarea>

- Atribut Maxlength
    Atribut ini berfungsi untuk memberikan batasan jumlah karakter yang dapat diinputkan kedalam textarea. 
    Contoh :
    <textarea rows="5" cols="20" maxlength="25"></textarea>

- Memberikan Inputan Awal
    Kita dapat memberikan inputan awal pada textarea dengan cara menulis kalimat tersebut diantara tag pembuka dan penutup dari element textarea.
    Contoh :
    <textarea rows="5" cols="20">Inputan Awal</textarea>

PERTEMUAN 61 : Select, Option, dan Optgroup Element

Element Select
    Elemen <select> digunakan untuk membuat daftar drop-down. Elemen <select> paling sering digunakan dalam formulir, untuk mengumpulkan input pengguna.
    Contoh : 
    <select></select>

    - Atribut Name
        Atribut ini diperlukan untuk mereferensikan data formulir setelah formulir dikirimkan (jika Anda menghilangkan atribut nama, tidak ada data dari daftar drop-down yang akan dikirimkan).
        Contoh : 
        <select name="makanan">
        </select>

    - Atribut id 
        Atribut ini diperlukan untuk mengaitkan daftar drop-down dengan label.
        Contoh :
        <select name="makanan" id="makanan">
        </select>
    
    - Atribut Multiple
        Dengan atribut ini, kita dapat memilih lebih dari 1 menu. Untuk memilih menu lebih dari satu, pengguna hanya perlu menekan tombol CTRL + Menu yang dipilih.
        Contoh :
        <select name="makanan" id="makanan" multiple>
            <option value="soto-kudus">Soto Kudus</option>
            <option value="tude-bakar">Tude Bakar</option>
        </select>

Element Option
    Tag <option> mendefinisikan opsi dalam daftar pilih. Elemen <option> masuk ke dalam elemen <select>, <optgroup>, atau <datalist>.
    Contoh :
    <select name="makanan" id="makanan">
        <option value="soto-kudus">Soto Kudus</option>
        <option value="tude-bakar">Tude Bakar</option>
    </select>

    Catatan: Tag <option> dapat digunakan tanpa atribut apa pun, tetapi Anda biasanya memerlukan atribut 'value', yang menunjukkan apa yang dikirim ke server pada pengiriman formulir.

    - Atribut Selected
        Dengan atribut ini, kita bisa menandai menu mana yang dipilih sejak awal website diakses.
        Contoh :
        <option value="tempoyak-ikan" selected>Tempoyak Ikan</option>

Element Optgroup
    Tag <optgroup> digunakan untuk mengelompokkan opsi terkait dalam elemen <select> (daftar drop-down). Jika Anda memiliki daftar opsi yang panjang, grup opsi terkait lebih mudah ditangani oleh pengguna.
    Contoh :
    <select name="pesanan" id="pesanan">
        <optgroup label="Makanan">
            <option value="soto-kudus">Soto Kudus</option>
            <option value="tude-bakar">Tude Bakar</option>
            <option value="tempoyak-ikan">Tempoyak Ikan</option>
            <option value="asinan-betawi">Asinan Betawi</option>
            <option value="kerak-telor">Kerak Telor</option>
            <option value="nasi-padang">Nasi Padang</option>
        </optgroup>
        <option label="Minuman">
            <option value="es-jeruk">Es Jeruk</option>
            <option value="es-teh">Es Teh</option>
            <option value="es-campur">Es Campur</option>
        </optgroup>
    </select>

    - Atribut label
        Atribut label menentukan label untuk <optgroup>.

PERTEMUAN 62 : Datalist Element

Datalist Element adalah sebuah element yang berfungsi menyediakan list data untuk dijadikan referensi pada element input text, sehingga ketika kita mengisi element input text akan ada saran / suggestion sesuai dengan datalist yang kita buat.
Contoh :
<datalist id="listmakanan">
    <option value="ayam-goreng">Ayam Goreng</option>
    <option value="ayam-bakar">Ayam Bakar</option>
    <option value="sate-padang">Sate Padang</option>
    <option value="sate-madura">Sate Madura</option>
    <option value="nasi-padang">Nasi Padang</option>
</datalist>

Untuk menghubungkan element datalist dengan element input text yang dimasuksud, diperlukan atribut lain yaitu 'id' sebagai identifier pada element datalist, dan 'list' pada element input sebagai referensi datalist yang akan digunakan.
Contoh :
<form>
    <input type="text" list="listmakanan">
</form>
